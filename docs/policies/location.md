# 장소 검색 정책

## 감지 키워드
```
맛집, 카페, 음식점, 병원, 약국, 주변, 근처, 위치, 가는 길
```

## 감지 조건
- 메타데이터에 `address` 존재
- 쿼리에 장소 키워드 포함

## 데이터 추출
| 필드 | 설명 |
|------|------|
| name | 장소명 |
| address | 주소 |
| phone | 전화번호 |
| rating | 평점 |
| category | 업종 |
| openHours | 영업시간 |
| imageUrl | 장소 이미지 |

## 템플릿
- **메인**: `map`
- **컨트롤러**: `distance`, `category-filter`, `sort`

## 예시 쿼리
| 쿼리 | 처리 |
|------|------|
| 강남 맛집 | map 템플릿 |
| 근처 카페 | map 템플릿 |
| 서울역 가는 길 | map 템플릿 |
