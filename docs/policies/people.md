# 인물 검색 정책

## 감지 조건
- `disp-attr="PRF"` (인물 프로필 컴포넌트) 존재
- 쿼리가 한글 2~4글자 이름 패턴 (`/[가-힣]{2,4}$/`)
- 쿼리 리라이트에서 `people` 의도 감지

## 쿼리 확장
인물 + 이슈 조합 시 이중 쿼리로 확장:

| 원본 | 확장 |
|------|------|
| 이병헌 대종상 | "이병헌"(people) + "이병헌 대종상"(news) |
| 손흥민 토트넘 | "손흥민"(people) + "손흥민 토트넘"(news) |

## 데이터 추출 (LLM)
| 필드 | 설명 |
|------|------|
| name | 이름 |
| occupation | 직업 |
| organization | 소속 |
| birthDate | 생년월일 |
| imageUrl | 프로필 이미지 |

## 동명이인 처리
이중 쿼리 병합 시 LLM이 동일인 여부 판단:
- 직업/분야 다르면 제외 (가수 vs 바둑기사)
- 소속 다르면 제외
- 맥락 다르면 제외

## 템플릿 분기
| 조건 | 템플릿 |
|------|--------|
| 인물 1명 + 뉴스 | hero |
| 인물 2명 + 뉴스 | dual-profile |
| 인물만 1명 | profile |
| 인물 여러 명 | grid |

## 예시 쿼리
| 쿼리 | 처리 |
|------|------|
| 손흥민 | profile 또는 hero |
| 이병헌 대종상 | hero (인물+뉴스) |
| BTS 멤버 | grid (여러 인물) |
