# 일반 검색 정책 (기본값)

## 적용 조건
특정 쿼리 타입에 해당하지 않는 경우

## 템플릿 선택 기준
| 조건 | 템플릿 |
|------|--------|
| 이미지 포함 + 6개 이상 | gallery |
| 아이템 3개 이상 | hero |
| 그 외 | card |

## 데이터 추출
| 필드 | 설명 |
|------|------|
| title | 제목 |
| description | 설명 |
| url | 링크 |
| imageUrl | 이미지 (있으면) |
| category | 카테고리 |

## 컨트롤러
- `sort`: 정렬
- `filter`: 필터

## 예시 쿼리
| 쿼리 | 처리 |
|------|------|
| Next.js 튜토리얼 | hero 또는 card |
| 맥북 사용법 | hero 또는 card |
