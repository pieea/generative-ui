# 쇼핑 검색 정책

## 감지 키워드
```
추천, 가격, 최저가, 구매, 쇼핑, 비교, 리뷰, 평점, 할인
```

## 감지 조건
- `disp-attr="SNY"` (쇼핑 컴포넌트) 존재
- 메타데이터에 `price` + `rating` 동시 존재
- 쿼리에 쇼핑 키워드 포함

## 데이터 추출
| 필드 | 소스 | 설명 |
|------|------|------|
| title | 상품명 | 필수 |
| price | 가격 | 필수 |
| rating | 평점 | 선택 |
| reviewCount | 리뷰 수 | 선택 |
| seller | 판매처 | 선택 |
| imageUrl | 상품 이미지 | 선택 |
| url | 상품 링크 | 필수 |

## 템플릿
- **메인**: `shopping`
- **컨트롤러**: `price-range`, `sort`, `filter`

## 예시 쿼리
| 쿼리 | 처리 |
|------|------|

| 에어팟 최저가 | shopping 템플릿 |
